# Support

For bugs, layout problems, accessibility issues, or confusing flows, open a GitHub issue and include:

- Browser and device.
- Whether TalkType was installed as a PWA.
- Transcription mode: Live Text, After Stop, or Offline.
- Steps to reproduce.
- Screenshots when useful.

Please do not include private transcripts, recordings, API keys, supporter codes, payment details, or anything you would not want public.

For security concerns, use [SECURITY.md](SECURITY.md). For commercial licensing or supporter/payment questions, contact the maintainer through the GitHub profile links.
